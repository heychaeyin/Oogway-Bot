Data Exploration

Data sets

For our data sets, we had scavenged a full data set of around 21K+ emotional phases from 3 data sets in total which each data set file had contained a large table. This table we had to go through to get only the names of every emotion and every quote in a corresponding order separated by a ';'. We had then added such to a text file where such file was fed into our AI text model to be trained on which took roughly about a hour. We had successfully accomplished this for all 3 data sets we had gotten for our model, originating from Red it. This data set is known as "Go Emotions" where we had ignored the rest of the given data for such data that was provided for the breakdown of emotions relating them to negative or positive type of emotional analysis.For our second hand data set, we had classed each emotion given in the first data set then used Open AI to generate 10 emotional phrases, some of which became slightly biblical in nature when quoting from people such as Winston Churchill, Albert Einstein, and even one from Jesus Christ oddly enough. These quotes were then given the same treatment as the first data set though it was easier considering we were inputting quotes for specific emotions. This second hand data set we called the "fortune cookie data set", we had split the emotion to which we manually applied into the emotion it corresponds to and the quote. As explained previously or in the future of this project, this emotion it corresponds to is then matched with the emotional analysis of the user input. If the emotion of the quote matches, the fortune cookie quote is then printed to the user as a form of advice.



Full Corpus Word Map

To the left is a word map built using all the words in our training data corpus. Word maps like these help us visualize the relative frequency of each word. For example, the words "name" and "love" appear frequently in the training data set so they appear larger than the other words. Other emotion labels like admiration, approval, and anger are also fairly large. These words are a subset of the training labels so we expect them to be present frequently.

Emotions Bar-Chart

The bar chart gives us a visual-numerical representation of the frequency with which emotions occur in the training data. Originally, neutral emotions dominated the data set biasing our language model towards labeling statements as neutral. Randomly removing a proportion of these observations produced a less biased model.

3D-PCA Word Embedding Visualization

Using a pretrained Word2Vec model, vectorized words were plotted in Euclidean 3-space. By converting words to vectors we can conceptualize distance and angle in non-euclidean space. Words that are contextually more similar should appear near one another in the grid. For example, we can see that approve and approved are one another's nearest neighbors (bottom left). Similarly, joy, passion, happiness, loving, and spirit are plotted in the same cluster.
