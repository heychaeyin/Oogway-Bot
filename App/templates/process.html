The Journey

About the Project

This project we all collectively named the Oogway Bot was created with the intent of creating a AI bot that will provide emotional and philisophical support to the user based on sentiment analysis of the user's input based on given input data from a Reddit/Go Emotions dataset that we saw fit. On our secondhand, we got a much smaller dataset from OpenAI where we asked it for emotional quotes which followed suite with the GoEmotions emotion list. Thereby generating emotional responses from user input, we extracted the emotion from which is then matched with every single quote we got from OpenAI If any quote matched the emoitional analysis of the user input phrase, that quote or quotes are then added to a list where the computer prints out a random index of that list and gives you the quote you desire.
